# Renalo

Renalo is a private, self-hosted budgeting workspace for tracking accounts, income, expenses, transfers, recurring entries, and balance adjustments.

## Documentation

For installation, configuration, administration, security guidance, and complete feature walkthroughs, visit the [Renalo user guide](https://renalo-docs.orange-buffalo.io/).

Developer and domain references are maintained in [`docs/developer`](docs/developer/README.md), including the [development guide](docs/developer/Development.md) and [financial calculation contract](docs/developer/FinancialCalculations.md).

## Container Images

Versioned release images are published to GitHub Container Registry:

- Application: `ghcr.io/orange-buffalo/renalo:<version>`
- User guide: `ghcr.io/orange-buffalo/renalo-docs:<version>`

Use an explicit version from [GitHub Releases](https://github.com/orange-buffalo/renalo/releases). Deployment instructions are available in the [installation guide](https://renalo-docs.orange-buffalo.io/getting-started/installation/).
