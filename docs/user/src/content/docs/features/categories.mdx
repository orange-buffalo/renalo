Renalo keeps expense and income categories separate. This avoids mixing concepts such as groceries and salary while still giving each overview meaningful filters.

## Archive instead of delete

Archived categories remain visible in Budget settings but disappear from new transaction forms and filters. Existing transactions remain linked and readable.

## Merge duplicates

When two categories mean the same thing, merge the source into another category of the same type. Renalo re-links every matching transaction and deletes the source.

<ScreenshotPair desktop={mergeDesktop} mobile={mergeMobile} alt="Expense category merge form" />

The merge action is hidden when no other active category is available. Category merges cannot be undone.
