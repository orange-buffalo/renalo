Transfers connect two accounts without pretending that money was earned or spent. The source side contributes outflow and the target side contributes inflow on the dashboard.

<ScreenshotPair desktop={desktop} mobile={mobile} alt="Funds transfer overview" />

## Same currency

Enter one amount. Renalo stores the same minor-unit value on both sides.

## Different currencies

Enter the actual amount leaving the source and arriving at the target. The exchange-rate field helps calculate one from the other, but the two account amounts are authoritative.

<ScreenshotPair desktop={formDesktop} mobile={formMobile} alt="Cross-currency transfer form" />

The displayed rate means one source-currency unit equals the shown target-currency amount. Renalo rounds the derived target to the target currency's smallest unit and rejects unsafe values rather than silently losing precision.

Renalo also uses those authoritative transfer amounts as evidence when representing foreign-currency income and expenses in your default account's currency for analytics. For example, if foreign income is later transferred in full to the default currency, its comparison value is the amount that actually arrived. Renalo does not substitute an external average market rate when there is no matching transfer.
