## Core calculation policy

- Money is stored as integer minor units in each account's currency.
- Current balances include posted activity through the current UTC date, inclusive.
- Future activity is planned and does not affect today's balance.
- Cross-currency transfer sides retain separate amounts; Renalo does not combine currencies into one total.
- Financial arithmetic fails on integer overflow instead of silently wrapping.

The complete implementation contract is in `docs/developer/FinancialCalculations.md` in the source repository.
